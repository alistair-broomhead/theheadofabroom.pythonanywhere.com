My experiments

Below are a set of experiments that run in the browser.

Chalk And Cheese

Chalk And Cheese is a simple multiplayer card game - it's not ready yet.

Particle Exploder

Exploder was a simple exercise to polish my understanding of JavaScript's prototypical inheritance model while learning to make use the html5 canvas element. I took simplistic code from an online tutorial, here, deconstructed it, and then put it back together as a set of objects that allowed for extension, such as dynamic resizing and support for multi-touch.

Message

Message was an experiment in putting together a micro-framework for generating html content - I wanted to see wht I could achieve while minimising external dependencies to just bottle. While somewhat successful, the context management approach I followed with this framework did not seem as clean or elegant as relying on jinja2, as I have for the rest of this site.
